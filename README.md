NxTest
======

An energy efficiency test tool for TV online

![效果图](http://github.com/Garfielt/NxTest/raw/master/NxTest.png)


Runtime：

Python 2.7
Wxpython
Pyserial


使用方法：

电路串口连接Chroma 7120（可更改串口通信部分适配其他彩色分析仪）
输入机编按空格键开始测试

计算公式 Eff = (L*S)/(Pe-10)/1.1

layout类似的话可更改部分代码后借用移植使用.

采用BSD协议发布，可按需求修改使用。
